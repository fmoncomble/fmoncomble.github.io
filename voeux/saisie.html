Mode d'emploi

  1. Enseignements (à gauche) :

    Pour ajouter un enseignement à vos vœux, sélectionnez-le à l'aide des menus déroulants, puis cliquez sur Ajouter à côté de sa description.

    • Pour dupliquer un enseignement, cliquez sur le ➕.

    • Pour le supprimer, cliquez sur le ❌.

    • Le volume horaire de votre service est ajusté en temps réel : en orange s'il est insuffisant, vert s'il est conforme, rouge s'il dépasse la limite autorisée.

  2. Indisponibilités (à droite) :

    • Dans le tableau, cliquez sur vos plages horaires d'indisponibilité pour les sélectionner ❌.

    • Cliquez à nouveau sur une case pour la désélectionner ✅.

  3. Valider :

    • Quand vous avez terminé, cliquez sur Valider, vérifiez le récapitulatif et cliquez sur Confirmer pour créer votre fichier de vœux.

    • Un fichier JSON est téléchargé : envoyez-le conjointement à Florent Moncomble et Guillaume Winter.
